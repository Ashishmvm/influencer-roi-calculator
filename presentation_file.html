📐 ROI Calculation Formula

ROI = ((Revenue - Investment) / Investment) × 100

Where:

  • Revenue = Estimated Sales × Product Price (₹)
  • Investment = Total Influencer Costs (₹)
  • Estimated Sales = (Total Reach × Conversion Rate²) / 100

⚙️ Installation & Setup

🚀 Quick Start Guide

# Clone the repository
git clone https://github.com/yourusername/influencer-roi-calculator.git

# Navigate to project directory  
cd influencer-roi-calculator

# Install dependencies
npm install

# Start development server
npm start

# Open browser to http://localhost:3000

📋 Prerequisites

  • Node.js: Version 14.0 or higher
  • npm/yarn: Package manager
  • Modern Browser: Chrome, Firefox, Safari, Edge
  • Git: For repository management

🔧 Build for Production

  • Build Command: npm run build
  • Output Directory: build/
  • Deployment: Static hosting compatible
  • Performance: Optimized for production
